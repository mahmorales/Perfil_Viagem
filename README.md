# Travelgram - Perfil de Viagens

## Descrição

Este é um projeto de website estático que apresenta o perfil de viagens de Isabela Torres. Inspirado no conceito de redes sociais como o Instagram, o Travelgram permite compartilhar histórias e fotos de viagens de forma simples e elegante. O site destaca a paixão por viagens, cultura e gastronomia, com foco em experiências registradas em São Paulo e ao redor do mundo.

## Funcionalidades

- **Perfil do Usuário**: Exibe informações pessoais, localização (São Paulo, Brasil), estatísticas de viagens (37 países visitados e 240 fotos).
- **Galeria de Fotos**: Uma seção principal com imagens de viagens organizadas em um layout responsivo.
- **Navegação**: Menu superior com opções para explorar, minhas viagens e perfil.
- **Design Responsivo**: Utiliza CSS para garantir uma boa visualização em diferentes dispositivos.

## Tecnologias Utilizadas

- **HTML5**: Para a estrutura e semântica do conteúdo.
- **CSS3**: Para estilização, layout e responsividade, incluindo fontes do Google Fonts (Poppins).

## Como Visualizar o Projeto

1. Clone ou baixe este repositório para o seu computador.
2. Navegue até a pasta do projeto.
3. Abra o arquivo `index.html` em qualquer navegador web moderno.

Não há necessidade de servidor ou dependências adicionais, pois é um site estático.
